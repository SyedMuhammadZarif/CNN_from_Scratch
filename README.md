<h2>MNIST Digit Classification with PyTorch</h2>

<p>
A small practice project built while revising PyTorch fundamentals. This project trains a simple Convolutional Neural Network (CNN) on the MNIST handwritten digit dataset and serves as a hands-on exercise for understanding the core PyTorch workflow.
</p>

<h3>What I Practiced</h3>

<ul>
  <li>Loading datasets with TorchVision</li>
  <li>Computing dataset normalization statistics</li>
  <li>Building a CNN using PyTorch modules</li>
  <li>Implementing training and evaluation loops</li>
  <li>Tracking loss and accuracy metrics</li>
  <li>Saving trained model weights</li>
</ul>

<h3>Model Architecture</h3>

<ul>
  <li>Conv2D (1 → 8) + ReLU + MaxPool</li>
  <li>Conv2D (8 → 16) + ReLU + MaxPool</li>
  <li>Fully Connected Layer (10 classes)</li>
</ul>

<h3>Purpose</h3>

<p>
This project was created as a revision exercise to reinforce fundamental PyTorch concepts, including data preprocessing, model construction, training, evaluation, and model persistence.
</p>
